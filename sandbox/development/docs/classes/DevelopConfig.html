DevelopConfig Class

Defined in: development/js/src/stack/develop/DevelopConfigProvider.js:3
Module: stack.develop.DevelopConfig

The DevelopConfig provides adopting implementations with a 'develop' configuration option. Depending upon the implementation, it is sometimes useful to understand whether or not the application is in a 'develop' or 'production' mode. DevelopConfig provides the .setDevelop(flag) method and should be configured within the compiled app.html page.

See /clientserver/views/app.html for example.
